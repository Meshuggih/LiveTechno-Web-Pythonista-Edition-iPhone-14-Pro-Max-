# Moog Subsequent 37

## Vue d'ensemble

**Type** : Synthé analogique monophonique  
**Fabricant** : Moog Music  
**Modèle** : Subsequent 37  
**Année** : 2017  

## Caractéristiques

- **Oscillateurs** : 2 VCO + 1 sub-oscillateur
- **Filtres** : Ladder filter 4-pôles (24 dB/oct)
- **Enveloppes** : 2 ADSR
- **LFO** : 1 LFO
- **Effets** : Distortion, delay
