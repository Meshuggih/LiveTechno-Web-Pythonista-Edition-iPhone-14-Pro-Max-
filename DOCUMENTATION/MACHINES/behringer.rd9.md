# Behringer RD-9

## Vue d'ensemble

**Type** : Boîte à rythmes analogique  
**Fabricant** : Behringer  
**Modèle** : RD-9  
**Année** : 2020  

## Caractéristiques

- **Voix** : 11 voix analogiques (kick, snare, toms, hats, cymbales, clap, rim)
- **Séquenceur** : 16 pas, 64 patterns
- **Sorties** : Mix + sorties individuelles
- **MIDI** : In, Out, Thru
